「修仙」是一个无服务器、只在本地运行的网络小说阅读器，为学习软件编程开发，分享出来有需要的朋友可以自行下载体验

「水印相机」以前是「修仙」的一个小功能，可以随意修改水印文字、地址、经纬度等信息
#### 功能：
- 1、网络小说搜索、本地TXT小说导入、TXT小说导出
- 2、阅读器支持自定义颜色样式、更换字体、翻页动画
- 3、阅读器支持排版切换：横排、竖排、左起、右起
- 3、内置Edge TTS引擎(参考代码：https://github.com/wangz-code/ddo-tts)
- 4、内置广告拦截网页浏览器
- 5、支持内部数据库导出、导入可以方便备份，可设置数据库位置为外部存储
- 6、有需要可自行下载体验
#### 「修仙」的优势：
##### 1、AI友好
「修仙」的自有书源，使用Python脚本编写，书源可在Windows独立运行，可由AI完成书源错误修复和新书源编写，[书源制作教程](https://github.com/SJJ-dot/Reader/blob/BookSource/test/README.MD)
##### 2、兼容阅读3.0书源规则
「修仙」不仅支持自有的Python书源并且同时兼容阅读3.0书源规则
