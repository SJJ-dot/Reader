### 书源规则参考 

想要自制书源的朋友请参考书源模板，选择合适的模板复制改改就行。书源调试可使用test模块（test模块需要作为项目根目录直接打开），已设置好测试环境

[默认书源格式参考：https://github.com/SJJ-dot/Reader/blob/master/test/BookSource/default.json](test/BookSource/default.json)
